# Toggle Elytra Mod

[English](README.md) | 日本語

プレイヤーの状態に応じてエリトラとチェストプレートを自動で切り替え、滑空中の花火使用も補助する Minecraft Fabric 用クライアントサイド mod です。

## 機能

* **チェストプレート自動装備**: 地上にいるとき、エリトラを装備していればインベントリ内のチェストプレートを自動で装備します。
* **空中トグル**: 空中で **ジャンプキー** を押すと、エリトラとチェストプレートを切り替えます。
* **バニラ通りの滑空開始**: 空中でエリトラをすでに装備していて、まだ滑空していない場合は、**ジャンプキー** でチェストプレートへ戻さずそのまま滑空を開始します。
* **花火の自動使用**: 滑空中に空手で右クリックすると、インベントリ内の花火を一時的にホットバーへ入れて使用し、元に戻します。
* **メニュー中は無効**: インベントリや各種メニュー画面を開いている間は、エリトラ切り替えと花火処理を行いません。
* **クライアントサイド専用**: サーバー側への導入なしでマルチプレイでも動作します。

## 必要環境

* Minecraft Java Edition **1.21.11**
* Fabric Loader
* Fabric API

## 導入方法

1. 最新のリリース `.jar` ファイルをダウンロードします。
2. Minecraft 1.21.11 用の [Fabric Loader](https://fabricmc.net/use/installer/) を導入します。
3. ダウンロードした `.jar` ファイルと [Fabric API](https://modrinth.com/mod/fabric-api) の `.jar` を `.minecraft/mods` フォルダへ入れます。
4. Fabric プロファイルで Minecraft を起動します。

## 使い方

1. **地上**: 地上を歩くか立っていると、インベントリにチェストプレートがありエリトラを装備している場合、自動で切り替わります。
2. **空中トグル**: 高所から飛び降りて、空中で **Space** を押すとエリトラとチェストプレートを切り替えます。
3. **滑空開始**: 空中ですでにエリトラを装備している場合は、**Space** を押すとバニラ通りに滑空を開始します。
4. **花火**: 滑空中に空手で右クリックすると、インベントリ内の花火を使用します。

## ソースからビルド

```bash
./gradlew build
```

出力された jar は `build/libs/` に生成されます。
